# Monthly Challenges Using Django
### Description

Monthly Challenges is a lightweight Django web application that showcases month-wise challenges along with an automatically calculated breakdown of weekdays and weekends. The project demonstrates clean URL routing, simple view logic, and responsive template-based UI using core Django features.


### Features

- Displays challenges for all 12 months

- Clean and readable URL structure

- Dynamic calculation of:

- Total days

- Weekdays

- Weekend days

- Contextual insights based on weekend count

- Django template inheritance with a shared base layout

- Responsive UI for desktop and mobile



### Technologies Used

- Python

- Django

- Django Templates

- HTML

- CSS

### How to Use
1. Clone the Repository

1. Run the Development Server
````
  python manage.py runserver
````
1. Open in Browser
````
  http://127.0.0.1:8000/
````
### Explore

- View all months on the home page

- Click any month to see its challenge and weekday/weekend breakdown
